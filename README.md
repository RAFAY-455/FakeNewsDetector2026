# 📰 Fake News Detection System

A Machine Learning based Fake News Detection System developed using Python and Flask. This project predicts whether a news article is **Real** or **Fake** using a trained machine learning model.

---

## 📌 Features

- Detects Fake and Real News
- Flask Web Interface
- Machine Learning Model
- Fast Prediction
- User-Friendly Interface

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Pandas
- NumPy
- Pickle

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/RAFAY-455/FakeNewsDetector2026.git
```

### Move into Project Folder

```bash
cd FakeNewsDetector2026
```

### Install Required Libraries

```bash
pip install flask pandas numpy scikit-learn
```

### Run the Project

```bash
python Source_Code_VS_Code/app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
